if (argc != 4) {
    printf("what?\n");

这里说明要有4个参数（包括程序名在内）

unsigned int first = atoi(argv[1]);
if (first != 0xcafe) {
    printf("you are wrong, sorry.\n");
    exit(2);
}

这里说明第二个参数的十六进制形式为cafe，十进制形式是51966。（atoi()是字符转整数的意思，是直接转，即"134213"转到134213）

if (second % 5 == 3 || second % 17 != 8) {
