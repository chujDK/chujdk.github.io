Middle of nowhere

Posted on Jan 1, 1

Oops! 404..

There is nothing in this page

好吧，还是有的。熟悉我的朋友知道以前我的博客不长这样。的确，以前是基于 typecho 的动态博客，一直托管在我的阿里云服务器上。考虑到阿里云不是我家的，所以租的服务器当然会过期。

应该是在 2021 年过期过一次，那个时候还有学生机可以买。不过比较麻烦的是，学生机改成了所谓的老云翼计划，不能直接续费，需要购买新的产品（从 ecs 服务器变成了轻量应用服务器），不能直接转移数据。当时我还在非常积极地编博客，所以不怕麻烦搞了几天转完了。说起来那个时候还很菜，数据库和 web 什么的都完全不懂，能转好还是挺神奇的。

然后在 2023 年的八月，又过期啦！这个时候的我还是很菜，该懂的还是都不懂，所以懒得不敢再去转移了。另一方面，虽然很菜，但是我一直想要一个和大佬一样的博客，简洁一些，一看就知道这个人很牛逼，不像我之前那个，花里胡哨的，一看就是小学生。所以我把文章全部考了下来，使用 hugo 作为生成器，基于 Archie 主题，加以一些魔改，通过 github pages 进行托管。好啦，现在博客不用花钱维护了，而且更重要的是，我也可以伪装成大佬了。

由于各种原因，博客的更新非常缓慢。而现在我也开始打工了，可能之后的更新频率只会更低。不过还是感谢各个时常打开我博客看看有没有更新的朋友。也感谢从各种途径搜索到这个博客的新朋友，希望能给大家分享一些知识，让大家有所收获。

最后还要向之前评论了这个博客的朋友道个歉，作为静态博客，评论只能基于一些第三方插件实现，这个博客采用了 giscus ，我没搞懂怎么把 typecho 的评论导入到这里面。所以之前的评论都消失了，感谢你们之前愿意给小站一些评论，给网站增添了一些生气，也让我有了更多写下去的动力。

还要向你的浏览器道个歉，没搞明白 github pages 怎么自定义 404 页，添加了 404.html 后，404 也会变成 200 了，你的浏览器可能对此不太满意。

都看到这里，真的不随便评论一下吗？能评论的 404 页多酷：）
